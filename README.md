Classic memory game inspired by the original Simon Game.

The player must repeat an increasingly long sequence of colors and sounds. :red_circle: :large_blue_circle: :green_circle: :yellow_circle:

<br/>



## General Information

This project recreates the classic Simon memory game in the browser.
The goal is to test and improve memory skills by repeating randomly generated sequences.
The project was created for learning and practicing JavaScript logic and DOM manipulation.

## Features

- Random sequence generation
- Increasing difficulty level
- Sound effects for each button
- Game over screen with restart option
